# gomono

Gomono is a boilerplate for SaaS or fullstack web application. it is designed with simple clean architecture - modular monolith. 

## Features

### Auth
- [] login 
- [] JWT Token
- [] Forgotten password

### Users
- [] shipped with four default access level: superadmin, admin, client, guest

### Payments
- [] Stripe integration

### Profiles
- [] Upload personal avatar
- [] Update bio
- [] set new email

### CRUD
- [] database integration (postgres by default)
- [] sample entity
